return和break及continue

发表于 2018-03-14 | 分类于 java

return

return关键字有两方面的作用：

  • 指定一个方法返回什么值，假设它没有void返回值；
  • 会导致当前方法退出，并返回那个值。
    如果在返回void的方法中没有return语句，那么在该方法的结尾处会有一个隐式的return，因此在方法中并非总是必须要有一个return语句。

break和continue

break用于强行退出循环，不执行循环中剩余的语句。而continue则停止当前迭代，然后退回循环起始处，开始下一次迭代。
